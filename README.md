# Web Scheduling System

* Repository for the system developed during the Hero's Week course.

* Scheduling system for a beauty salon

## Technologies and frameworks:

Client: 
  * React, 
  * ContextAPI, 
  * ModuleCSS, 
  * Day Picker

Server: 
  * Node, 
  * Express, 
  * JWT, 
  * Prisma, 
  * SQLite

## To execute the project:

``` bash
yarn install
```
```bash
yarn dev
```
